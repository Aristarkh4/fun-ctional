# `prepend`

[home](../README.md) &gt; [`prepend`](http://ramdajs.com/docs/#prepend)

Returns a new list containing the contents of the given list, preceded by the given element.

```js
import { prepend } from 'ramda'

prepend('x', ['y', 'z'])   // returns ['x', 'y', 'z']
```
