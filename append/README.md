# `append`

[home](../README.md) &gt; [`append`](http://ramdajs.com/docs/#append)

Returns a new list containing the contents of the given list, followed by the given element.

```js
import { append } from 'ramda'

append('z', ['x', 'y'])   // returns ['x', 'y', 'z']
```
